# UBI Brand Assets

Upload your brand assets to this folder organized by type:

## Recommended Structure

```
brand/
├── logos/           # Main logos (SVG, PNG)
├── icons/           # App icons, favicons
├── images/          # Hero images, backgrounds
├── fonts/           # Custom fonts (if any)
└── colors.json      # Brand color palette (optional)
```

## Naming Conventions

- **Logos**: `logo-{variant}.{ext}` (e.g., `logo-full.svg`, `logo-icon.png`, `logo-white.svg`)
- **Icons**: `icon-{size}.png` (e.g., `icon-192.png`, `icon-512.png`)
- **Favicons**: `favicon.ico`, `favicon.svg`

## Supported Formats

- **SVG** - Preferred for logos and icons (scalable)
- **PNG** - For raster images with transparency
- **ICO** - For favicons
- **WOFF2/WOFF** - For web fonts
